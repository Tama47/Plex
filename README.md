# Plex

### Plex Media Player ([Download Here](https://www.plex.tv/media-server-downloads/#plex-app))
#### Location for Mac:
```bash
~/Library/Application Support/Plex/
```
#### Location for Windows:
```bash
%LOCALAPPDATA%\Plex
```

%LOCALAPPDATA%\Plex Media Server
